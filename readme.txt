软件: FindESP
作用: 
根据传进来的分区盘符, 若磁盘是gpt分区, 则返回该盘符所在磁盘的esp分区盘符, 若磁盘是mbr分区, 则不返回
平台: C++ + VS2015 + win10_x64_build10586
作者: Bluer
时间: 2015.12.03

Software: FindESP
How it uses: 
input the drive letter, if the kind of partition table is gpt, it outputs esp partition letter, or it outputs nothing.
